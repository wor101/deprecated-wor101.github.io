# Overview:
Sinatra app for managing character interactions in RPGs

A simple CRUD app using a PostgreSQL database to manage interactions between characters (non-player characters & player characters).

## Developed With:
* Ruby
* Sinatra
* PostgreSQL
* BCrypt
